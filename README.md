# Branch Design Challenge.
> The most challenging part was making the AI voice agent not automatically end the conversation and prompting it to ask more questions before the user was done asking questions. I also discovered that the AI can seamlessly go from one conversation branch to another when given the prompt to do so from the user.

# Knowledge Base Impact.
> The voice agent's responses acted like the store in question was GameStop, given a lot of the resources mentioned GameStop in question, seeing how that was the most relevant store mentioned. This type of content certainly helped more than others as it gave it more of an identity and allowed it to pull from greater sources.

# Real-world Application
> This can be used as a way to help people with simple questions or direct people to actual helpers, making it so a voice agent does the basic questions, which would help productivity. Generally customer service is the one job field this voice agent would help severely within. Given that it mainly serves the role of an information booth.
